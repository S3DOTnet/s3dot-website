# S3DOT 文章改善案(Hero / Service / CTA)

作成日: 2026-07-19
根拠資料: [docs/CONTACT_CONVERSION_ACTION_PLAN.md](./CONTACT_CONVERSION_ACTION_PLAN.md) の「1. 修正推奨」1-1・1-2

本ファイルは文章案のみを提示するものであり、コード変更は行っていない。掲載する「改善案」はあくまで提案であり、最終的な文言は反映前にユーザー確認を得ること。

## 前提条件

- S3DOTの高級感・未来感を維持する
- 「AIを、もっと身近にする会社」というブランド軸を維持する
- 営業感を強くしすぎない
- 中小企業経営者にも理解できる文章にする
- 既存デザイン(構造・レイアウト・コンポーネント構成)の変更は前提にしない — 既存のテキスト要素の中身を差し替える提案のみとし、新しい要素の追加は提案しない

## 1. Heroセクション([HeroSection.tsx](../src/components/sections/HeroSection.tsx))

### 現状の文章

| 要素 | 現状 |
| --- | --- |
| ラベル([HeroSection.tsx:159-170](../src/components/sections/HeroSection.tsx#L159-L170)) | `AI PARTNER · S3DOT` |
| メイン見出しH1([HeroSection.tsx:173-199](../src/components/sections/HeroSection.tsx#L173-L199)) | `AIは特別なものじゃない。もう、仕事のスタンダードです。` |
| サブコピー([HeroSection.tsx:214-236](../src/components/sections/HeroSection.tsx#L214-L236)) | `使わない方がコストも時間も、確実に高くつく時代へ。S3DOTは、AIを「使える力」に変え、会社の成長を加速させます。` |
| プライマリCTA([HeroSection.tsx:244-259](../src/components/sections/HeroSection.tsx#L244-L259)) | `まずは無料で相談する` |
| セカンダリCTA([HeroSection.tsx:262-276](../src/components/sections/HeroSection.tsx#L262-L276)) | `公式LINEで相談する` |

### 課題

- サブコピーは「AIを使わないと損をする」という危機意識には訴えるが、「何から始めればいいか分からない」という読み手の状態そのものへの言及がなく、ターゲット(AI導入に迷う中小企業経営者)が自分ごと化しにくい。

### 改善案

**H1(メイン見出し)について**

H1は`SOCIAL_IMAGE.alt`([site-metadata.ts:9](../src/lib/site-metadata.ts#L9))など他ファイルでも同一文言が使われているサイトの主要タグラインであるため、**本提案では変更を提案しない**。変更する場合は影響範囲(OGP画像のalt文言等)の確認が別途必要になる。

**サブコピー**

- Before:
  > 使わない方がコストも時間も、確実に高くつく時代へ。S3DOTは、AIを「使える力」に変え、会社の成長を加速させます。

- After(案):
  > 「何から始めればいいか」は、まだ分からなくていい。使わない方が、コストも時間も確実に高くつく時代へ。S3DOTは、AIを「使える力」に変え、会社の成長を加速させます。

  - 冒頭に読み手の状態(何から始めればいいか分からない)を一言添えることで自分ごと化を狙う一方、既存の「使わない方が高くつく時代」という一文(FAQセクション等でも使われている一貫したメッセージ)と、ハイライト表示される「AIを「使える力」に変え、会社の成長を加速させます。」の部分はそのまま残し、既存トーン・デザイン(強調スタイル)への影響を避ける。

**CTAボタン文言**

- プライマリ「まずは無料で相談する」、セカンダリ「公式LINEで相談する」は、いずれも押しつけがましさがなく、既にContactSectionの主要CTAとも文言が一致しているため、**現状維持を推奨**する。

## 2. Serviceセクション([ServiceSection.tsx](../src/components/sections/ServiceSection.tsx))

現状、各カードの説明文はやや抽象的で、業種・具体例が伝わりにくい。`body`テキストのみを対象に、具体例を一言加える案を提示する(`icon`・`tags`・カード構造は変更しない)。

### 1. AI導入相談

- Before([ServiceSection.tsx:18](../src/components/sections/ServiceSection.tsx#L18)):
  > AIについて知識がなくても大丈夫。現在の業務・課題・目標をヒアリングし、あなたの会社に本当に合った活用方法を一緒に見つけます。

- After(案):
  > AIについて知識がなくても大丈夫。建設業の見積もり対応から士業の書類作成まで、業種を問わず現在の業務・課題・目標をヒアリングし、あなたの会社に本当に合った活用方法を一緒に見つけます。

### 2. 業務改善

- Before([ServiceSection.tsx:26](../src/components/sections/ServiceSection.tsx#L26)):
  > 既存の業務フローを壊すことなく、AIを自然に組み込みます。人がやらなくていい作業をAIに任せ、人は人にしかできない仕事に集中できる環境を作ります。

- After(案):
  > 既存の業務フローを壊すことなく、AIを自然に組み込みます。例えば経理の伝票整理や総務の問い合わせ対応など、人がやらなくていい作業をAIに任せ、人は人にしかできない仕事に集中できる環境を作ります。

### 3. 自動化

- Before([ServiceSection.tsx:34](../src/components/sections/ServiceSection.tsx#L34)):
  > 毎日同じことを繰り返している作業を洗い出し、自動化の仕組みを構築します。データ収集・集計・転記・通知など、まるごと自動で動かします。

- After(案):
  > 毎日同じことを繰り返している作業(日報の集計、受発注の入力など)を洗い出し、自動化の仕組みを構築します。データ収集・集計・転記・通知など、まるごと自動で動かします。

### 4. AI制作

- Before([ServiceSection.tsx:42](../src/components/sections/ServiceSection.tsx#L42)):
  > AIを活用した画像・動画・文章・音声コンテンツの制作を支援します。広告バナー、SNS投稿、商品説明文など、制作コストと時間を大幅に削減できます。

- After(案):
  > AIを活用した画像・動画・文章・音声コンテンツの制作を支援します。飲食店のSNS投稿から小売店の商品説明文まで、制作コストと時間を大幅に削減できます。

### 5. AIシステム開発

- Before([ServiceSection.tsx:50](../src/components/sections/ServiceSection.tsx#L50)):
  > 既存ツールでは対応できないケースのために、オーダーメイドのAIシステムを構築します。設計・開発・運用までワンストップで対応します。

- After(案):
  > 既存ツールでは対応できないケース(独自の受発注システムとの連携など)のために、オーダーメイドのAIシステムを構築します。設計・開発・運用までワンストップで対応します。

### 補足

- いずれも文末に短い具体例を挟む形に留め、文章量を大きく増やさないことで、カードの高さ・グリッドの見た目(高級感のある余白バランス)への影響を最小限にしている。
- 挙げた業種例(建設業・士業・経理・総務・飲食店・小売店等)は説明用の例示であり、実際に対応実績があるという意味ではない。掲載する場合は「活用イメージ」の範囲であることを踏まえた表現にする。

## 3. CTA(無料相談導線)

対象となる主なCTA文言は以下の通り。いずれも既存要素の文言のみを対象とし、新規ブロックの追加は提案しない。

| 箇所 | 現状の文言 |
| --- | --- |
| [Header.tsx:153](../src/components/layout/Header.tsx#L153)(PC常時表示) | `無料相談` |
| [Header.tsx:221](../src/components/layout/Header.tsx#L221)(スマホメニュー) | `無料相談` |
| [HeroSection.tsx:252](../src/components/sections/HeroSection.tsx#L252) | `まずは無料で相談する` |
| [ContactSection.tsx:218](../src/components/sections/ContactSection.tsx#L218) | `まずは無料で相談する` |
| [price/page.tsx:285](../src/app/price/page.tsx#L285) | `無料で相談してみる` |
| ContactSectionの選択カード([ContactSection.tsx:16-25](../src/components/sections/ContactSection.tsx#L16-L25)) | `無料相談(フォーム)` / `相談フォームへ` |

### 課題

- Header・Hero・ContactSectionでは「まずは無料で相談する」に統一されているが、料金ページのみ「無料で相談してみる」と語尾が異なる。表現自体はどちらも営業感が強くなく問題はないが、サイト全体としての文言統一という観点で軽微な揺れがある。

### 改善案

- 料金ページのCTA文言を、他ページと同一の表現に統一する。

  - Before([price/page.tsx:285](../src/app/price/page.tsx#L285)):
    > 無料で相談してみる

  - After(案):
    > まずは無料で相談する

  - 変更はテキストのみで、ボタンの形状・色・配置([price/page.tsx:281-287](../src/app/price/page.tsx#L281-L287))には影響しない。

### 現状維持を推奨する箇所

- Header・Hero・ContactSectionの「まずは無料で相談する」「無料相談」、LINE導線の「公式LINEで相談する」は、簡潔で押しつけがましさがなく、条件(営業感を強くしすぎない)に合致しているため変更不要。
- ContactSectionの選択カード文言(「相談フォームへ」「メールを送る」「LINEで相談する」)も同様に、行動が明確で過度な誘導表現がないため現状維持を推奨する。

## まとめ

本提案は、Hero・Service・CTAいずれも既存のテキスト要素の中身を差し替えるだけで実施可能であり、コンポーネント構造・色・余白・アニメーションなど既存デザインには変更を加えない。特にH1見出しはサイト全体で参照される主要タグラインのため変更対象から除外し、サブコピー・カード説明文・CTA文言の語尾統一に絞って提案した。反映する場合は、各文言についてユーザーの最終確認を経ること。
